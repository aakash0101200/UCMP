# UCMP - College Dashboard

A modern college dashboard project built with React, Vite, and modern web technologies.

## Overview

UCMP is a web-based college dashboard application designed to help students manage their academic information and activities in one centralized location.

## Technology Stack

- **Frontend**: JavaScript (92.4%), CSS (7.5%), HTML (0.1%)
- **Build Tool**: Vite
- **Framework**: React
- **Styling**: CSS

## Features

- Intuitive dashboard interface
- Real-time data display
- Responsive design
- Fast refresh during development

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aakash0101200/UCMP.git
```

2. Navigate to the project directory:
```bash
cd UCMP
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

The application should now be running on `http://localhost:5173`

### Building for Production

To create a production build:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
